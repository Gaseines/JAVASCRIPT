Converter para Número

Para números inteiros usar : Number.parseInt(n)

Para números reais usar: Number.parseFloat(n)

Modo simplificado usar: Number(n)

Formatação de número

Para casas decimais usar: n1.toFixed(2)

1545.5
> 1545.50

Para trocar dados usar: n1.toFixed(2).replace('.', ',')

1545.50
>1545,50

Para colocar cifrão usar: n1.toLocaleString('pt-BR', {style: 'currency', currency: 'brl' })

1545.5
> R$ 1.545,50

JSON

JSON = JavaScript Object Notation

Utilizado para comunicação entre servidores, ex: back end <-> front end

Basicamente um tipo de dado padronizado, que lembra muito um objeto do JavaScript

Exemplo:
{ "name": "Gabriel",
"age": 21,
"position": "Developer",
"languages": ["PHP", "JavaScript", "Python"]
}

Precisa seguir rigorosamente o padrão do JSON

Apenas aspas duplas (exceto em Number)

Não aceita comentarios

Para acessar os itens do JSON, usa-se a mesma forma do objeto

Converter de JSON para String

let pessoa = {
"nome": "Gabriel",
"idade": 21,
"profissao": "programador" }

let pessoaTexto = JSON.stringify(pessoa) Aqui converte de JSON para String

let pessoaJSON = JSON.parse(pessoaTexto)Aqui converte de String para JSON
