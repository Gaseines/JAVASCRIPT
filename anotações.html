Números

Converter para Número

Para números inteiros usar : Number.parseInt(n)

Para números reais usar: Number.parseFloat(n)

Modo simplificado usar: Number(n)

Formatação de número

Para casas decimais usar: n1.toFixed(2)

1545.5
> 1545.50

Para trocar dados usar: n1.toFixed(2).replace('.', ',')

1545.50
>1545,50

Para colocar cifrão usar: n1.toLocaleString('pt-BR', {style: 'currency', currency: 'brl' })

1545.5
> R$ 1.545,50

Concatenação

Para concatenação padrão usa-se ' aspas simples e com +. Exemplo:
'Estou aprendendo' + s

Para o uso de "abreviações" (template string) didita-se com ` crase e ${}. Exemplo:
`Estou aprendendo ${s}`

Propriedades

Adiciona e Remove Itens

let carro = { marca: "Honda", portas: 4, manual: true, motor: 1.7, velocidade: "Rapido" }

delete carro.velocidade Deleta a informação de "velocidade"

carro.tetoSolar = false Adiciona a informação de "tetoSolar"

console.log(carro.tetoSolar) Mostra a informação de "tetoSolar"

Copia de um Objeto para outro

let obj1 = { nome: "Gabriel", idade: 21 }

let obj2 = { carro: "Honda Civic" }

Object.assign(obj1, obj2) Copia as propriedades do "obj2" para o "obj1"

Resultado: let obj1 = { nome: "Gabriel", idade: 21, carro: "Honda Civic" }
