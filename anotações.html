Anotações

Números

Converter para Número

Para números inteiros usar : Number.parseInt(n)

Para números reais usar: Number.parseFloat(n)

Modo simplificado usar: Number(n)

Formatação de número

Para casas decimais usar: n1.toFixed(2)

1545.5
> 1545.50

Para trocar dados usar: n1.toFixed(2).replace('.', ',')

1545.50
>1545,50

Para colocar cifrão usar: n1.toLocaleString('pt-BR', {style: 'currency', currency: 'brl' })

1545.5
> R$ 1.545,50

String

Converter para String

String(s)
ou
n.toString(s)

Quantidade de caracteres

s.length

MAIÚSCULA

s.toUpperCase()

minúsculas

s.toLowerCase()

Concatenação

Para concatenação padrão usa-se ' aspas simples e com +. Exemplo:
'Estou aprendendo' + s

Para o uso de "abreviações" (template string) didita-se com ` crase e ${}. Exemplo:
`Estou aprendendo ${s}`

Operadores

Aritiméticos

Soma: +
Subtração: -
Multiplicação: *
Divisão Real: /
Divisão Inteira: %
Potencia: **

Lógicos

! negação 1º
&& conjunção 2º
|| disjunção 3º

idade >= 15 && idade <=17 // a idade está entre 15 e 17?

estado == 'RJ' || estado == 'SP'// o estado é RJ ou SP?

salário > 1500 && sexo != 'M'// salário é maior que 1500 e é mulher?
